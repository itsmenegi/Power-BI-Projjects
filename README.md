# E-Commerce Sales Dashboard (Power BI)

## Project Overview

This project is a Power BI dashboard that analyzes e-commerce sales performance.
It helps understand sales trends, product performance, and customer behavior.

## Tools Used

* Power BI
* Excel Dataset
* Data Visualization

## Dashboard Insights

* Total Sales and Profit
* Top Performing Products
* Regional Sales Analysis
* Monthly Sales Trends

## Files Included

* E-Commerce Sales Dashboard.pbix
* Dataset (Excel)
* Dashboard screenshot
